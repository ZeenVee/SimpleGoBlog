# Simple Blog using Golang

## API Endpoint
###Create an article
- Method: `POST`
- Path: `/articles`

### Get article by ID
- Method: `GET`
- Path: `/articles/<article_id>`

### Get all articles
- Method: `Get`
- Path: `/articles`

## Packages Used
- Dependency management: `dep`
- Database: `MySql`
- Routing: `chi`
